Centre d'aide.

Apprenez à utiliser Kroma Invoice comme un pro en quelques minutes.

1. Configuration du Wallet

Pour utiliser Kroma Invoice, vous avez besoin d'un wallet compatible Web3 (Metamask, Rabby, Rainbow, etc.).

  • Installez l'extension de votre choix.
  • Ajoutez le réseau Kroma Mainnet via Chainlist.
  • Connectez-vous sur l'application.

2. Créer votre première facture

Une fois connecté, cliquez sur le bouton "Nouvelle Facture". Vous devrez remplir :

  • Vos informations (Nom, adresse, logo).
  • Les informations de votre client.
  • Le détail des prestations (Description, Quantité, Prix).
  • Le token de paiement (USDT, USDC, KRO, ETH).
